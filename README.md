# Enterprise DevSecOps Framework
This repository demonstrates a production-ready CI/CD pipeline with "Shift-Left" security. It automates code quality checks, container vulnerability scanning, and policy enforcement.

## Features
- **Static Analysis:** SonarQube integration for code quality.
- **Container Security:** Trivy scanning for CVEs in Docker images.
- **Infrastructure Policy:** OPA (Open Policy Agent) to enforce cloud guardrails.
- **Secret Management:** Integration with AWS Secrets Manager/Vault.

## Tools
GitHub Actions, Docker, Trivy, SonarQube, OPA Rego.
